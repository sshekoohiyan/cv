# عضویت‌‌ها

- عضو TUV اتریش، مسئول پایداری، ایران  
- عضو مرکز توسعه آموزش (EDC) دانشگاه تربیت مدرس، تهران، ایران  
- عضو انجمن علمی بهداشت محیط ایران (IAEH)، ایران  
- عضو کمیته تحقیقات دانشجویی دانشگاه تربیت مدرس، تهران، ایران (۲۰۱۳–۲۰۱۷)  
- عضو مرکز تحقیقات مهندسی بهداشت حرفه‌ای و محیط‌زیست، دانشگاه علوم پزشکی هرمزگان (۲۰۱۳)  
- عضو مرکز تحقیقات ارتقای سلامت، دانشگاه علوم پزشکی هرمزگان (۲۰۱۲–۲۰۱۳)  
- عضو کمیته تحقیقات دانشجویی، دانشگاه علوم پزشکی هرمزگان (۲۰۱۱–۲۰۱۳)
